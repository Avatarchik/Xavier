## Xavier
A networking library based on [Telepathy](https://www.github.com/vis2k/telepathy)

## Introduction
Xavier builds upon Telepathy to provide tools to use network calls easily. Features include:
- `Node` instances that let you use `Client` and `Server` in an event oriented manner
- `Transit` that provides the following
    - `Publish/Subscribe` event system with payload support
    - `Request/Response` RPC like features to "invoke" code on another machine
